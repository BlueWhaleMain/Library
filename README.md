# Library
Minecraft数据包&资源包

修改原版机制使游戏规则趋于常理化，添加新机制拓展原版内容

添加东方内容，尝试在Minecraft中实现符卡规则

添加管理员工具，比命令方块更方便

## 注意
包含非原创内容，包内文档有详细介绍

弹幕造成的卡/炸服问题，见Wiki

由于是后期开源，在此不提供历史版本及其release，详见[发布网站](http://genghelidemc.sxl.cn/)
