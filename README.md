修改原版机制使游戏规则趋于常理化，添加新机制拓展原版内容

已添加的外来内容：

- TouhouProject
- Botania
- Undertale

弹幕造成的卡/炸服问题，见Wiki

由于是后期开源，在此不提供历史版本及其release，详见[发布网站](http://bamboomc.top/datapacks/)
